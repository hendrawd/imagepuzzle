Welcome to imagepuzzle!

The project aims to implement a [sliding puzzle](https://en.wikipedia.org/wiki/Sliding_puzzle) game, accessed through a web browser.
### Technologies used:
* HTML
* JavaScript
* CSS
* jQuery
